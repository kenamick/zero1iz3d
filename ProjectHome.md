# Zero1 IZ 3D #

Zero1\_IZ3D is a 3D game engine attempt using **C++/OpenGL**.

Features:

  * 3DS Model loader
  * Billboards
  * Particles
  * Fog

# Demo controls #

  * w,a,s,d - move around
  * u,i - light on/off
  * n,m - fog on/off
  * b - change fog color
  * o,p - mountains on/off
  * k,l -  grid on/off
